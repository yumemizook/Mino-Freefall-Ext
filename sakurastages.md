This page lists all 27 stages present in ''Tetris The Grand Master 3 Terror-Instinct''<nowiki>'</nowiki>s Sakura mode. Minos marked as [[File:CTet.png]] are the jewels which must be cleared to complete the stage.

Legend:
C = target block that needs to be cleared
I/T/O/S/Z/J/L = color to use for the preset pieces

. = empty space
