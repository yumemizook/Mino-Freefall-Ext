=== Konoha ===
{{main|Tetris The Grand Master 4 Absolute Eye/Konoha Characters}}
	
[[Image:TGM4_Konoha_mode.jpeg|200px|thumb|right|TGM4 Konoha mode screenshot]]

Konoha is a mode centered around big [[tetromino|tetrominoes]] and [[All Clear|All Clears (bravoes)]]. Successfully completing All Clears rewards players with illustrations, which can be viewed in the Konoha Character Gallery. 

==== Gameplay ==== 
The goal of this mode is to get as many Bravoes as possible in Big mode under a time limit. This time starts at 2 minutes for Standard Type, and 4 minutes and 10 seconds for TGM Type.

Extra time is obtained by performing certain clears. The timer has an upper limit of 5 minutes.

==== Kita Kitsune (Konoha{{efn|name=konoha}}) ====
In the game, Kita Kitsune will appear beside the playfield if the CPU detects an All Clear sequence is possible, celebrates when it's achieved or falls over when the opportunity is missed. It should be noted that it is not error-free and may miss some possible All Clears.  

An icon of Kita Kitsune in a flower appears below the playfield if the current character illustration is one that has not yet been unlocked. 

==== Assists ====
There is an optional assist which shows outlines of where to place pieces to achieve the All Clears that the CPU detects. The assist defaults to being turned on, and there is no penalty for using it. The assist mode goes away after the 25th All Clear in a run.

==== Easy and Hard difficulties ==== 
One can select between Easy and Hard mode by using Left or Right when Konoha is hovered on in the mode select.

Easy features only L, J, I, T and O pieces. Gravity and timings do not change over the whole course, and Easy mode ends when 110 all-clears are achieved. You only get access to the first 37 Konoha girls in Easy mode.

Hard mode features all seven pieces and continues indefinitely until the time runs out. Clearing more than 110 All clears (inclusive?) in hard mode awards the player the title KONOHA-MASTER (Km).
